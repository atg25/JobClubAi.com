# Personas

## 1. AI-Curious Freshman

- **Background:** First-year NJIT student, exploring tech and AI for the first time.
- **Goals:** Learn about AI, join events, build a basic portfolio, connect with peers.
- **Pain Points:** Overwhelmed by options, unsure where to start, lacks professional assets.

## 2. Career-Switching Senior

- **Background:** Senior in a non-CS major, wants to pivot into AI/tech before graduation.
- **Goals:** Build a strong LinkedIn/GitHub, attend workshops, get mentorship, land an internship/job.
- **Pain Points:** Lacks network, needs guidance on portfolio, feels behind peers.

## 3. Entrepreneurial Student

- **Background:** Ambitious student building an AI startup or side project.
- **Goals:** Find collaborators, attend networking events, get feedback on ideas, access resources.
- **Pain Points:** Hard to find like-minded peers, needs exposure, wants to showcase work.
