# Customer Journey Map

## Persona: AI-Curious Freshman

1. **Hears about Job Club**
   - Emotion: Curious, hopeful
   - Opportunity: Clear value proposition on homepage
2. **Visits website**
   - Emotion: Overwhelmed, but interested
   - Opportunity: Simple navigation, clear CTAs
3. **Explores events/resources**
   - Emotion: Motivated, but unsure where to start
   - Opportunity: Featured guides, beginner events
4. **Registers for onboarding**
   - Emotion: Excited, a bit anxious
   - Opportunity: Friendly onboarding form, checklist
5. **Receives personalized checklist/email**
   - Emotion: Supported, guided
   - Opportunity: Actionable steps, links to resources
6. **Attends first event**
   - Emotion: Included, engaged
   - Opportunity: Welcoming community, Discord invite
7. **Builds portfolio, connects with mentors**
   - Emotion: Confident, empowered
   - Opportunity: Ongoing support, advanced resources
